# 📖 About the Project

This project was developed to explore the practical application of **Machine Learning** in building intelligent recommendation systems.

Inspired by streaming platforms like Netflix, the application analyzes movie metadata—including genres, cast, crew, keywords, and overviews—to recommend movies with similar content using **Content-Based Filtering** and **Cosine Similarity**.

Beyond the recommendation engine, this project also focuses on creating a clean and interactive user experience through a modern Streamlit interface while integrating real-time movie information using the **TMDB API**.
